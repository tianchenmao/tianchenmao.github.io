Projects

China Robot Competition UAV Challenge

In the UAV Challenge, drones are required to complete tasks such as autonomous flight, localiziation, trajectory planning, target identification and delivering.The main rule is mentioned below.

  • Indoor:GPS is not avaliable,uav has to locate itself using camera or lidar
  • Identification:Five Targets with different scores to identify
  • Delivery:3 standard boxes need to be unloaded on target
  • Obstacle avoiding:Obstacles will be randomly placed in ground

My work:

  • Writing Logical Part:I am in charge of flow control, mainly write a finite state machine, which constantly switchs from different task or action.Common task like takeoff, navigation, searching for target, tracking target, unload object, which is built on basic actions like adjust position and velocity.
  • Writing Navigation Part: I am in charge of motion planning of drone.Our UAV is equiped with realsense D435i(RGB-D camera) or rplidar A2/A3/S2. So I use ego-planner for visual navigation, movabase (choosing teb as local planner) for lidar navigation.I write an interface to easily incorporate different planner into flowcontrol. I tune the parameter of planner, and add some additional behavier to gain better performance.
  • Building docker images: To facilitate migration and deployment, we use docker to launch our program. I build coresponding images on x86 and arm64 platform with all essential components in it. I also use shell script to update image quickly and use docker-compose to launch all container at once. Moreover, the image contains vnc server, so we can using web browser to access system.

A Web-site Platform for Drone Control

To speed up testing and tuning process, we developed a web-side drone control platform.The front-end is besed on Vue, and the back-end is based on ROS, using grpc to communicate with each other.User can setting taskID,navigation point and other parameter using web browser.

My work:

I am responsible for encapsulating the underlying behavior of the aircraft to facilitate front-end call selection.Communicate with the front-end code through GRPC, pass the received messages to the ROS node, and write a interface to receive action messages at the control node.

Simulation

Simulation of fixed wing aircraft based on simulink.Build the dynamic model of aircraft and trim it on straight level fight using linear analyse. And choose proper feedback to build longitudinal controller using simplified dynamics.

Simulation of quadcopter based on simulink.Build cascaded PID for position and attitude control.

Face Recognition

I use FaceNet to extract features vector from images and use kd-tree beasd knn to match a new face with those in database(choose a proper threshold of similarity of two face vectors). I using weights trained on CASIA-WEBFACE and test it on LFW.
